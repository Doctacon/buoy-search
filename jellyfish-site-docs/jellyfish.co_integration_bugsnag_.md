# Bugsnag Integration

Ingest Bugsnag error events to monitor application stability and production issue load.

#### Overview

Bugsnag is an error monitoring platform used to capture crashes and application errors across environments.

#### What this integration powers:

- Visibility into production error volume and stability trends across teams and services
- Reliability reporting by surfacing monitoring signals alongside delivery and work data
- Operational insights to help teams prioritize remediation and reduce production issue load

# VictorOps Integration

Sync VictorOps incidents and alerts to track interruptions and incident response load.

#### Overview

VictorOps (now Splunk On-Call) is an incident response and on-call platform for routing alerts, managing escalations, and coordinating incident workflows.

#### What this integration powers:

- A unified incident and alert event stream in Data Hub for reliability reporting and trend analysis
- Interruption and incident workload insights, such as alert volume over time and time-to-close patterns
- More complete delivery context by pairing incident activity with your connected repos, deploys, and delivery data

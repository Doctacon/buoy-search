G2 ranks products using a combination of user review data and public online signals. Scores are calculated based on:

- G2’s Satisfaction algorithms
- User satisfaction
- Review quality
- Product growth (employee, web, and social)
- Year-over-year trends.

![Jellyfish G2 Spring 2026 Grid](https://test-jellyfish-co.pantheonsite.io/wp-content/uploads/2026/03/spring26-grid-light.png)

Products are categorized through G2’s standardized methodology. Ratings are unbiased, real-time and verified for authenticity.

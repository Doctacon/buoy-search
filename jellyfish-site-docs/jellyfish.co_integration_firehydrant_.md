# FireHydrant Integration

Sync FireHydrant incidents and on-call events to track interruptions and response effort.

#### Overview

FireHydrant is an incident management platform that helps teams coordinate response, manage on-call workflows, and capture incident timelines.

#### What this integration powers:

- Incident records in Data Hub based on FireHydrant incident events and lifecycle updates
- Interruption and response effort trends, using incident timelines to understand frequency and time-to-close patterns
- Better cross-signal analysis when viewed alongside delivery data like code changes, deployments, and work items

# Claude Code Integration

Assess Claude Code adoption and outcomes to quantify AI-assisted productivity gains.

#### Overview

Claude Code is an AI coding assistant that helps developers generate, refactor, and understand code during day-to-day development workflows.

#### What this integration powers:

- AI Impact visibility into Claude Code adoption and usage trends by engineer and team
- Engagement insights like active usage days and usage patterns over time
- Reporting to evaluate AI tooling impact and guide productivity investment decisions

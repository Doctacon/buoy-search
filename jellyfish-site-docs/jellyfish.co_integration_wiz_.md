# Wiz Integration

Import Wiz cloud security findings to surface infrastructure risk and remediation work.

#### Overview

Wiz is a cloud security platform that detects risks across cloud environments, including misconfigurations, vulnerabilities, and runtime threats.

#### What this integration powers:

- Centralized intake of Wiz security events in Data Hub for consistent reporting
- Visibility into cloud risk trends across findings like issues, vulnerabilities, and detections
- Remediation tracking signals to understand risk reduction over time and where work is accumulating

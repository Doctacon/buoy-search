# Incident.io Integration

Sync incidents and response timelines to track interruptions and incident workload.

#### Overview

Incident.io is an incident management platform that helps teams coordinate response, manage status updates, and document outcomes.

#### What this integration powers:

- Incident records in Data Hub based on incident events, status changes, and closures
- Timeline and duration trends to understand interruption load and response patterns over time
- End-to-end context when viewed alongside delivery signals (work tracking, deploys, and code activity)

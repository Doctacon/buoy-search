# GitLab Integration

Sync repos, merge requests, and issues to connect delivery metrics to outcomes.

#### Overview

GitLab is a DevOps platform that helps teams manage source code, code changes, and issue workflows in one place. Jellyfish supports both GitLab SaaS and self-managed.

#### What this integration powers:

- Delivery visibility across dashboards like Metrics, Life Cycle Explorer, Deliverables, and Team Summaries
- Investment and work mapping through Allocations and operational reporting
- DevOps lead time metrics and broader delivery performance insights

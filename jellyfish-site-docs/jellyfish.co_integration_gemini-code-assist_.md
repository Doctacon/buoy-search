# Gemini Code Assist Integration

Measure Gemini Code Assist adoption and impact across developers and repos.

#### Overview

Gemini Code Assist is Google’s AI coding assistant that helps developers write and improve code within their development workflow.

#### What this integration powers:

- AI Impact insights on Gemini Code Assist adoption and usage by developer, team, and repo
- Usage trend reporting based on when logging was enabled
- Visibility to evaluate AI tooling impact and guide productivity investment decisions

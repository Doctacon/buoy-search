# CodeRabbit Integration

Assess CodeRabbit usage to measure AI code review coverage and efficiency.

#### Overview

CodeRabbit is an AI code review agent that reviews code changes and provides suggestions to help teams catch issues earlier and speed up reviews.

#### What this integration powers:

- AI Impact visibility into CodeRabbit adoption and activity trends across teams and repos
- Coverage insights that show how often CodeRabbit participates in reviews and where it is most used
- Quality signal analysis by comparing AI vs human review patterns to guide where AI review adds the most value

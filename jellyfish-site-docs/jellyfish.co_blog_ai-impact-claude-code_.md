Jellyfish [AI Impact](https://jellyfish.co/platform/jellyfish-ai-impact/) now supports [Claude Code](https://www.anthropic.com/claude-code) from Anthropic, an AI coding tool that gives developers the power to delegate substantial engineering tasks to Claude directly from their terminal. With Jellyfish and Claude, engineering leaders can better assess adoption, impact and productivity changes from AI over time.

![Jellyfish AI Impact dashboard_Claude Code](https://test-jellyfish-co.pantheonsite.io/wp-content/uploads/2025/08/Jellyfish-AI-Impact-dashboard_Claude-Code.png)

Claude works directly in VS Code and JetBrains, presenting suggestions directly in a user’s code files, creating a more integrated user experience. Jellyfish customers are already seeing the lift it provides to theirR&D teams:

> Claude Code has proven to be extremely useful for our Senior Engineers who perform larger and more complex tasks that require better contextual awareness.

_– Aleksandrs Dudarevs, Engineering Manager, Docuspace_

And now these customers will be able to systematically quantify its impact. Jellyfish’s deep integration with Claude Code provides joint customers the ability to:

- **Receive insights instantly:** Thanks to near real-time event-based data availability from the Claude Code API, Jellyfish customers can surface instant insights including adoption, usage, performance and more directly within the Jellyfish platform.
- **Measure Claude’s impact on overall developer productivity:** With AI Impact, you can directly quantify how Claude impacts various engineering workflows and use cases – from building new features to reducing technical debt, reviewing code quality, making security improvements and everything in between.
- **Scale Claude use across the engineering organization:** AI Impact pinpoints teams with high adoption to further accelerate deployment so you can replicate best practices and implementation strategies org-wide.

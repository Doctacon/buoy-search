# Snyk Integration

Import Snyk vulnerability data across dependencies, code, and containers to track security debt and fix rates.

#### Overview

Snyk is a developer security platform that scans open-source dependencies, first-party code, containers, and infrastructure as code to identify vulnerabilities and policy issues.

#### What this integration powers:

- Security event intake in Data Hub to centralize vulnerability signals
- Vulnerability trend reporting to track security debt over time
- Remediation progress visibility to understand fix momentum and backlog growth

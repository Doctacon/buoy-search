# TeamCity Integration

Import TeamCity build data to monitor release health and delivery metrics.

#### Overview

TeamCity is a CI/CD platform that automates builds, tests, and deployments across your software delivery pipeline.

#### What this integration powers:

- DevOps metrics like deployment frequency, lead time, change failure rate, and MTTR
- Release health visibility by surfacing build stability, failures, and delivery bottlenecks
- Delivery insights by connecting CI/CD activity to repos, teams, and initiatives over time

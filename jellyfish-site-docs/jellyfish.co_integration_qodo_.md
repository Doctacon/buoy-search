# Qodo Integration

Assess Qodo adoption and usage to quantify AI-assisted code review productivity gains across teams.

#### Overview

Qodo is a quality-first AI code review platform that helps engineering teams review, test, and maintain code integrity throughout the development lifecycle.

#### What this integration powers:

- AI Impact insights on Qodo adoption and code review activity by developer, team, and repo
- Visibility into review coverage trends, including PRs reviewed and review time improvements over time
- Reporting to evaluate AI code review ROI and guide productivity investment decisions

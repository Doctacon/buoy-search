# Unblocked Integration

Code review and analysis tool that provides feedback and insights directly on pull requests.

#### Overview

Unblocked is an AI code review and analysis tool that provides automated feedback on code changes to help teams identify issues faster and keep reviews moving.

#### What this integration powers:

- AI Impact visibility into Unblocked adoption and how consistently it participates in reviews across teams
- Workflow insights that show where Unblocked is helping reduce review back-and-forth and time-to-approval
- Adoption reporting to support rollout decisions and compare AI vs human review activity trends

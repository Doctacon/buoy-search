# Augment Integration

Evaluate Augment adoption and impact across teams to quantify AI-assisted development gains.

#### Overview

Augment Code is an AI coding assistant that meets engineers wherever they work – across the IDE, CLI, code review, and beyond. Jellyfish securely connects to Augment Code to ingest usage and telemetry data for AI Impact.

#### What this integration powers:

- AI Impact insights on Augment adoption and usage by developer and team
- Team-level visibility into where Augment is actively used and how usage trends over time
- Productivity reporting that connects Augment usage to key engineering workflows like feature development, code review, and technical debt reduction
- ROI measurement to help engineering leaders quantify the business impact of their AI investment and scale adoption with confidence

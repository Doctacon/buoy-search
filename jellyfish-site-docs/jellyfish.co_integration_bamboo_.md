# Bamboo Integration

Import build and deploy data to power DORA and release metrics.

#### Overview

Bamboo is Atlassian’s CI/CD platform for running build and deployment pipelines across your codebase.

#### What this integration powers:

- DevOps metrics like deployment rate, lead time, change failure rate, and MTTR
- Release reporting by linking deployments back to repos and changes
- Delivery and reliability insights to connect shipping cadence with stability trends

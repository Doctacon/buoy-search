# GitHub Copilot Integration

Evaluate Copilot usage and impact to quantify AI productivity across teams.

#### Overview

GitHub Copilot is an AI coding assistant that helps developers generate and complete code in their editor. Jellyfish securely connects to Copilot seat and usage data for each org or enterprise where Copilot licenses are managed.

#### What this integration powers:

- AI Impact insights on Copilot adoption and usage across developers and teams
- Seat visibility to understand where Copilot is enabled and actively used
- Productivity reporting to quantify how Copilot is changing development workflows

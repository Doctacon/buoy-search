# Sentry Integration

Ingest Sentry errors, crash reports, and performance issues to track production issues and reliability work.

#### Overview

Sentry is an application monitoring platform used to detect runtime errors, crashes, and performance issues across services.

#### What this integration powers:

- Reliability reporting by surfacing error and performance signals alongside engineering work data
- Visibility into production issue trends to help teams prioritize remediation work
- Operational insights by connecting monitoring events to broader delivery and performance reporting

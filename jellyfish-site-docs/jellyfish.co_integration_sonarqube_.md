# SonarQube Integration

Import SonarQube analysis to monitor code quality, coverage, and security trends.

#### Overview

SonarQube is a code quality platform that analyzes codebases to identify bugs, vulnerabilities, and maintainability issues.

#### What this integration powers:

- Metrics Explorer reporting for bugs, vulnerabilities, and code smells
- Coverage and technical debt metrics to track quality trends over time
- Quality gate performance to monitor pass rates and reliability/security ratings

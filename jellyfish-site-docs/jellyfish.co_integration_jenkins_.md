# Jenkins Integration

Ingest build and deployment signals to power DORA and release metrics.

#### Overview

Jenkins is an automation server used to run CI jobs and delivery pipelines across builds, tests, and deployments.

#### What this integration powers:

- DevOps metrics like deployment frequency, lead time, change failure rate, and MTTR
- Pipeline health reporting to spot slowdowns, failures, and stability trends over time
- Release visibility by tying deploy activity back to repos, teams, and delivery goals

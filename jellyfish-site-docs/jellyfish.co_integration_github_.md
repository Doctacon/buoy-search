# GitHub Integration

Sync repos, PRs, and issues to map work and measure cycle time.

#### Overview

GitHub is a source code management platform where teams collaborate on code changes and manage development workflows. Jellyfish supports connecting via direct connection or the Jellyfish Agent.

#### What this integration powers:

- Delivery visibility and workflows across dashboards like Metrics, Life Cycle Explorer, Deliverables, and Team Summaries
- Investment and work mapping through Allocations and operational reporting
- DevOps lead time metrics and other delivery performance insights

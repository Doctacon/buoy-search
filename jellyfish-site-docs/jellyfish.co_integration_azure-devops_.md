# Azure DevOps Integration

Connect repos, pipelines, and boards for full delivery visibility.

#### Overview

Azure DevOps (ADO) is Microsoft’s platform for planning work, managing source code, and running CI/CD pipelines. Connecting ADO to Jellyfish brings work items, delivery activity, and team context into one view.

#### What this integration powers:

- Delivery visibility: Connect work items and code changes to see progress, throughput, and bottlenecks across teams.
- Allocations and investment: Understand how effort maps to projects and priorities, then spot drift early.
- DevOps performance: Use pipeline and deployment signals to surface DORA-style delivery trends and release health.
- AI Impact readiness: Pair ADO context with AI tool usage to understand how AI is changing delivery outcomes and team performance.

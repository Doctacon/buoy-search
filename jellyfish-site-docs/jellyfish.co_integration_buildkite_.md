# Buildkite Integration

Ingest Buildkite build and deploy data to power DORA metrics and delivery insights.

#### Overview

Buildkite is a CI/CD platform that runs build, test, and deployment workflows on your own infrastructure.

#### What this integration powers:

- DevOps metrics like deployment frequency, lead time, change failure rate, and MTTR
- Release health visibility by surfacing build outcomes, failures, and pipeline stability
- Delivery insights by linking CI/CD activity to repos, teams, and initiatives over time

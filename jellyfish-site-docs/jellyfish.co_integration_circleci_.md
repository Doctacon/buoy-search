# CircleCI Integration

Stream build and deploy data to surface DORA metrics and release insights.

#### Overview

CircleCI is a CI/CD platform that automates build, test, and deployment workflows so teams can ship changes faster and more reliably.

#### What this integration powers:

- DORA and release metrics based on build and deployment activity
- Delivery visibility into release frequency, lead time, and change health trends
- Operational reporting to connect CI/CD performance with team delivery outcomes

# Codeship Integration

Capture build and deploy events to track speed, stability, and MTTR.

#### Overview

Codeship is a CI/CD platform for automating builds, tests, and deployments across applications and services.

#### What this integration powers:

- DevOps metrics like deployment frequency, lead time, change failure rate, and MTTR
- Release health reporting by tying deploy activity to repos and changes
- Delivery visibility to compare shipping cadence against stability trends over time

# Checkmarx Integration

Import Checkmarx scan results to track application security risk and remediation effort.

#### Overview

Checkmarx One helps teams identify application security issues through automated scanning and vulnerability detection.

#### What this integration powers:

- Centralized visibility into Checkmarx scan findings in Data Hub for security reporting and trend analysis
- Monitoring of security risk over time across apps and projects as new scans complete and findings change
- Clear tracking of remediation progress by showing how findings evolve across scan cycles

## Frequently Asked Questions

What is Jellyfish AI Impact and what does it measure?

Jellyfish AI Impact measures how AI tools, like Copilot, Cursor, Claude Code, and agentic systems, actually affect software delivery. It tracks adoption, usage, spend, and outcomes (speed, quality, productivity) to show which AI investments create real value.

How does Jellyfish measure the ROI of AI tools?

Jellyfish connects AI usage and spend to delivery metrics such as throughput, cycle time, code quality, and cost efficiency. This gives engineering leaders defensible ROI data instead of vendor-reported activity metrics.

Which AI tools and coding assistants does Jellyfish support?

Jellyfish is vendor-agnostic and supports tools like GitHub Copilot, Cursor, Claude Code, Amazon Q, Gemini Code Assist, Windsurf, CodeRabbit, and emerging agentic workflows such as Devin, Copilot Agent, and Google Jules.

Do we need to integrate every AI tool to get insights?

No. Jellyfish derives AI signals directly from Git, planning systems, and workflow data—so you get accurate adoption and impact insights without heavy integrations, tagging, or manual setup.

How fast can engineering teams see AI insights and value?

Most teams see adoption and spend insights within days, with delivery-impact trends emerging in the first sprint. Because Jellyfish analyzes existing Git and work data, setup is fast and insights appear quickly.

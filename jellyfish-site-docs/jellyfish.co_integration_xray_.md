# Xray Integration

Sync Xray test issues and results to connect QA coverage with delivery work.

#### Overview

JFrog Xray is a security and compliance scanning tool that analyzes artifacts, dependencies, and container images for vulnerabilities and policy violations

#### What this integration powers:

- Centralized reporting on security findings and policy violations over time, based on Xray event data
- Visibility into where security risk is surfacing (by watch/policy outcomes) to support prioritization and triage
- Tracking remediation workload and trendlines by analyzing violation frequency and severity signals as they evolve

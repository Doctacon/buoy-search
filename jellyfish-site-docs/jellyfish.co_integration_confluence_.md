# Confluence Integration

Sync Confluence activity to reflect documentation and planning work in performance metrics and allocations.

#### Overview

Confluence is a workspace for documentation, specs, and planning. Connecting Confluence to Jellyfish helps you capture engineering effort that happens outside code and tickets, so reporting better reflects the full scope of work.

#### What this integration powers:

- Documentation activity metrics: Quantify documentation work with counts of updates and Confluence “active days.”
- More complete allocations: Attribute time and cost to engineering work like planning, docs, and support tasks that often live in Confluence.
- Stronger work-to-context links: Associate Confluence pages to Jira issues to improve allocation views and delivery context.
- Smarter change signals: Use page content to distinguish meaningful edits from minor updates, improving the quality of documentation-based insights.

# GitHub Advanced Security Integration

Sync code scanning, secret scanning, and dependency review alerts to monitor security findings linked to Git activity.

#### Overview

GitHub Advanced Security helps teams detect and prioritize risks in code, dependencies, and secrets directly in GitHub.

#### What this integration powers:

- Centralized intake of GitHub security alerts in Data Hub for consistent security reporting
- Visibility into security findings by repo and organization (code scanning, secret scanning, dependency and Dependabot signals)
- Security trends tied to Git activity so teams can track where risk is showing up and how it changes over time

# New Relic Integration

Connect applied performance monitoring data to correlate development work with application performance.

#### Overview

New Relic is an observability and APM platform that monitors application health and performance.

#### What this integration powers:

- Reliability and performance signal tracking by bringing New Relic alert activity into Jellyfish for visibility over time
- Faster triage context by pairing monitoring signals with engineering work and delivery activity in Jellyfish
- Operational reporting on stability trends across services, environments, and alert policies

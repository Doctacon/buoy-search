# Argo CD Integration

Ingest Argo CD deployment events to measure release frequency and change failure risk.

#### Overview

Argo CD is a GitOps continuous delivery tool for Kubernetes that syncs application state from Git to clusters and manages rollouts.

#### What this integration powers:

- DevOps metrics like deployment frequency, lead time, change failure rate, and MTTR
- Release health insights by surfacing sync status, failed rollouts, and recovery patterns over time
- Delivery visibility by tying production delivery activity back to repos, teams, and initiatives

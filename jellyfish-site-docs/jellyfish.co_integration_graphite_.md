# Graphite Integration

Analyze Graphite-assisted code reviews to assess review cycle efficiency.

#### Overview

Graphite is an engineering workflow platform that includes AI-assisted code review to help teams review changes faster and more consistently.

#### What this integration powers:

- AI Impact reporting that separates Graphite agent activity from human reviews for clear adoption visibility
- Review cycle insights to understand where Graphite is reducing turnaround time and unblocking merges
- Comparative analysis of AI vs human review participation to guide rollout decisions and review policy

# Rootly Integration

Sync Rootly incidents and timelines to capture interruptions and incident response work.

#### Overview

Rootly is an incident management platform that helps teams run coordinated responses and capture structured incident timelines.

#### What this integration powers:

- Incident records in Data Hub based on Rootly incident events and timeline updates
- Interruption and response workload trends, including frequency and time-to-resolve patterns over time
- More complete reliability reporting when viewed alongside delivery signals like deploys, code changes, and work items

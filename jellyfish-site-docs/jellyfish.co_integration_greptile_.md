# Greptile Integration

Evaluate Greptile adoption to assess AI-assisted code review speed and efficiency.

#### Overview

Greptile is an AI code review agent that reviews code changes and surfaces issues and suggestions to help teams improve review quality and consistency.

#### What this integration powers:

- AI Impact visibility into Greptile usage over time and where it is being adopted across teams
- Review effectiveness insights by comparing AI vs human review participation and activity patterns
- Rollout and governance reporting to understand which repos or teams benefit most from AI-assisted review

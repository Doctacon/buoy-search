# Rollbar Integration

Import Rollbar error reports and frequency data to analyze production reliability and impact.

#### Overview

Rollbar is an error monitoring platform that captures exceptions and crash reports, helping teams understand what’s breaking in production and how often it happens.

#### What this integration powers:

- Production reliability visibility by surfacing error volume and frequency trends across monitored projects
- Operational insight into where instability is increasing, helping teams prioritize fixes and reduce recurring issues
- Performance reporting that brings monitoring signals into Jellyfish for broader delivery and team-level context

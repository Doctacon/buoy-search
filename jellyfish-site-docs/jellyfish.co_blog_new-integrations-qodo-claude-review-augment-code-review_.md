Jellyfish is excited to announce support for three new code review agents: Qodo, Claude Review, and Augment Code Review. With this release, thousands of engineering organizations already using the Jellyfish platform can surface and analyze activity from these tools directly within [Jellyfish’s AI Impact dashboard](https://jellyfish.co/platform/jellyfish-ai-impact/).

As AI-assisted coding becomes the norm across engineering organizations, the surge in output is putting increasing pressure on code review, often turning it into a bottleneck. AI review agents can help absorb that load and keep work moving, but they also introduce a new variable into the system. Engineering leaders need clear visibility into how these tools are performing and whether they’re actually improving review speed and quality, or simply adding noise to a critical process.

Jellyfish can provide that visibility be helping leaders answer questions like:

- Are teams actually adopting these tools?
- Are they improving review speed?
- Do developers find the feedback useful?

Along with these new integrations, Jellyfish integrates with a range of AI tools across that work software development lifecycle. You can view all [Jellyfish AI integrations here](https://jellyfish.co/integrations/).

## What You Can Measure

With these integrations, Jellyfish’s AI Impact dashboard now provides deeper visibility into both the adoption and effectiveness of an expanded range of code review agents, providing data on:

- **Overall Adoption:** How frequently each tool is used in code reviews
- **Adoption by repository:** Where usage is concentrated across your codebase
- **PRs reviewed:** The volume of pull requests receiving AI-assisted feedback

These metrics help engineering leaders identify where AI is gaining traction and where there may be opportunities to expand usage – or where it’s failing to provide sufficient improvement.

By integrating directly with industry leaders like Qodo, Claude, and Augment, we’re giving R&D teams the telemetry they need to prove that AI agents – and AI coding tools across the SDLC – are reducing cycle times, not just increasing PR volume.

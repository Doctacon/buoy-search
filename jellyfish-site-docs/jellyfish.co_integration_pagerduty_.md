# PagerDuty Integration

Sync incidents and on-call data to monitor interruptions and MTTR.

#### Overview

PagerDuty is an incident management platform for alerting, on-call, and incident response. Jellyfish ingests PagerDuty incident events to create incident records, then uses them to calculate reliability and recovery metrics in DevOps Metrics.

#### What this integration powers:

- Incident records in DevOps Metrics built from PagerDuty incident data (start time, resolve time, service labels, links)
- Reliability metrics like Failure Rate and Change Failure Rate based on incident outcomes
- Recovery performance tracking through Mean Time to Resolve (MTTR)

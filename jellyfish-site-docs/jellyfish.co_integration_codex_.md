# Codex Integration

Evaluate Codex adoption and impact to quantify AI-assisted developer productivity gains.

#### Overview

Codex is OpenAI’s coding agent that supports developers across the software development lifecycle, helping teams move faster, streamline workflows, and ship higher-quality software.

Jellyfish securely connects to Codex to ingest usage and telemetry data for AI Impact.

#### What this integration powers:

- AI Impact insights on Codex adoption and usage by developer and team
- Team-level visibility into where Codex is actively used and how usage trends over time
- Delivery impact reporting that ties Codex usage to outcomes across the SDLC, from feature velocity and code quality to cycle time and technical debt reduction
- ROI measurement to help engineering leaders quantify the business impact of their AI investment and scale adoption with confidence

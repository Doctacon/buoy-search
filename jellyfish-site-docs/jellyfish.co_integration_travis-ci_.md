# Travis CI Integration

Ingest Travis CI build and deploy activity to surface DORA metrics and delivery insights.

#### Overview

Travis CI is a CI/CD platform that automates builds, tests, and deployments so teams can ship changes consistently and safely.

#### What this integration powers:

- DevOps metrics like deployment frequency and lead time trends
- Release insights that show how often teams ship and how delivery patterns change over time
- Stronger reliability context when paired with incident and error tracking data, helping connect releases to recovery work

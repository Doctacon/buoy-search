## Insights to Improve Daily Work

![](https://jellyfish.co/wp-content/uploads/2025/12/feature-ai-code-ratio.webp)

### Track Workflow Shifts

See how the balance of AI-generated vs. human-generated code evolves to understand changing workflow patterns.

![](https://jellyfish.co/wp-content/uploads/2025/12/feature-Utilization-matrix-1.webp)

### Identify Tool Strengths

Compare how each tool performs across writing, refactoring, reviewing, documenting, and debugging to optimize task fit.

![](https://jellyfish.co/wp-content/uploads/2025/12/feature-ai-impact-mx-filter.webp)

### Analyze Workflows Deeply

Drill into AI’s impact by individual, team, role, and more to understand complex workflow changes and adapt accordingly.

![](https://jellyfish.co/wp-content/uploads/2025/12/feature-ai-impact-agent-reporting.webp)

### Understand Agent Collaboration

View PRs, contribution splits, and merge outcomes to learn how humans and agents collaborate, and which patterns drive success.

![](https://jellyfish.co/wp-content/uploads/2026/01/feature-code-quality.webp)

### Monitor Code Quality

Use quality metrics to see how AI affects code health and identify where workflows need adjustment.

![](https://jellyfish.co/wp-content/uploads/2025/12/feature-Developer-feedback-2.webp)

### Incorporate Developer Signals

Combine sentiment on enablement, productivity, and suggestion quality with system data for early, directional insight.

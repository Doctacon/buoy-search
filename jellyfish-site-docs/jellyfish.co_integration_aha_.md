# Aha! Integration

Sync initiatives and releases to align roadmaps with investments and track delivery progress.

#### Overview

Aha! Roadmaps connects with Jira, so teams can write initiative and release metadata back to Jira where it can flow into Jellyfish alongside delivery signals.

#### What this integration powers:

- Connect roadmap initiatives and releases to the Jira work teams execute every day
- Align engineering investment views to roadmap priorities and planning structure
- Track progress against initiatives with delivery context already in Jellyfish

# Bitbucket Integration

Sync commits and PRs to link code, issues, and initiatives.

#### Overview

Bitbucket is Atlassian’s source code management platform where teams collaborate on code changes and manage development workflows.

#### What this integration powers:

- Delivery visibility across Metrics, Deliverables, and team-level reporting
- Investment and work mapping through Allocations and operational views
- Cycle time and lead time insights to understand delivery throughput and bottlenecks

# Dynatrace Integration

Ingest Dynatrace performance and alert data to connect application health and reliability issues to engineering investment.

#### Overview

Dynatrace is an observability platform used to detect, prioritize, and troubleshoot production problems across applications, services, and underlying infrastructure.

#### What this integration powers:

- Data Hub visibility into Dynatrace problem notifications, including severity and lifecycle changes
- Better tracking of reliability and remediation workload by capturing when monitoring issues are detected, escalated, and cleared
- Stronger operational reporting by pairing Dynatrace problem signals with engineering activity to understand where reliability work is concentrating over time

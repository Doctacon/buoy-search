# TestRail Integration

Sync TestRail test runs and results to reflect QA activity in quality and release metrics.

#### Overview

TestRail is a test management platform used to plan, execute, and report on QA testing, including test cases, test runs, and results.

#### What this integration powers:

- Visibility into QA execution volume and trends based on test runs, result activity, and case updates
- Quality and release reporting that incorporates test outcomes alongside delivery signals
- Better understanding of QA workload over time by tracking when testing work is created, updated, and completed

# Sumo Logic Integration

Import Sumo Logic operational log and tracing data to monitor system performance and stability remediation workload.

#### Overview

Sumo Logic is a log analytics and observability platform used to monitor production systems, investigate incidents, and alert on operational issues.

#### What this integration powers:

- Centralized visibility into operational health signals by ingesting Sumo Logic alerts into Jellyfish
- Better tracking of stability and remediation workload by capturing when issues are detected and escalated
- Operational reporting on reliability trends across services and environments using alert activity over time
